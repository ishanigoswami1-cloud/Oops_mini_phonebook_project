# Python PhoneBook Project

## Description
This project is a simple PhoneBook application created using Python and Object-Oriented Programming (OOP).  
It allows the user to store contact names and phone numbers while validating the phone number before saving.

## Features
- Add a new contact
- Validate phone numbers
- Store contacts in a phone directory list
- Display a success message when a contact is saved

## How the Program Works
1. The user enters the contact name.
2. The user enters the phone number.
3. The program checks whether the phone number is valid (10 digits and numeric).
4. If the phone number is valid, the contact is saved in the phone directory.

## Example
Input:
Enter the name of the contact: Riya  
Enter the phone number: 9876543210  

Output:
Contact saved successfully

## Concepts Used
- Python Classes
- Object-Oriented Programming (OOP)
- Constructor (`__init__`)
- Class variables
- Input handling
- Basic validation logic
